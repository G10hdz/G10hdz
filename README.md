### Ergane — Job Search Automation Engine *(private)*
> Positronica Labs

AI-powered job search automation for Mexico/LATAM. Scrapes 8+ job boards with Playwright, matches CVs with semantic scoring via Claude API, generates tailored Word CVs, and sends Telegram notifications. Runs as a systemd daemon with interactive Telegram bot commands.

`Python` `Playwright` `Claude API` `SQLite` `APScheduler` `Telegram Bot API`

---

### [Metis](https://github.com/G10hdz/METIS) — Local Personal AI Agent
> Positronica Labs

AI Agent Orchestrator with LangGraph StateGraph and 9 routes (RAG, code, search, file ops, bash, general). Keyword-based router with 5-tier LLM fallback (Ollama → Opencode → Qwen → Gemini → Telegram). Full file operations, bash sandbox with whitelisted commands, and Telegram integration.

`Python` `LangGraph` `Ollama` `Telegram Bot API` `Gradio`

---

### Hypatia — Adaptive Learning for Neurodivergent Minds *(private)*
> Positronica Labs

Adaptive learning platform with OCR pipeline for exam prep materials. Processes PDFs (UNAM exam banks, study guides), chunks content, generates explanations, and stores in ChromaDB for RAG-based retrieval. Built for accessible education.

`Python` `OCR` `ChromaDB` `RAG` `JavaScript`

### [Echo](https://github.com/G10hdz/ECHO) — Pronunciation Practice App (ELSA-like)

`Python` `Whisper` `WhisperX` `Kokoro TTS` `LangGraph` `React` `FastAPI`
